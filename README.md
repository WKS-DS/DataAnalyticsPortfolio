## Projects

### [Titanic Survival Analysis](./Python/Titanic_Midterm_WScott_v2.ipynb)
A complete exploratory data analysis and narrative report examining survival patterns on the 1912 Titanic disaster. Uses four targeted visualizations to test the popular "women and children first" narrative and demonstrates how class and gender interacted to determine survival outcomes. Includes a full IMRaD technical report, ethics reflection, and an intentionally misleading visualization paired with a critique.

**Tools:** Python, Pandas, Seaborn, Matplotlib, Jupyter

### [HR Analytics Dashboard](https://public.tableau.com/app/profile/will.scott8637/viz/HR_Tableau_17823693965210/Q3)
Interactive Tableau dashboard analyzing employee demographics, salary distribution, and attrition patterns across a multi-department organization. Built on a three-table relational dataset with a star schema data model.

**Tools:** Tableau Public, multi-table data modeling, calculated fields

### [Sales Performance Dashboard](./PowerBI)
Power BI dashboard featuring KPI cards, interactive slicers, and detailed transaction-level visibility. Supports filtering by product category and customer type to analyze sales performance across regions, sales reps, and quarters.

**Tools:** Microsoft Power BI

### [Cryptocurrency Investor Communication Report](./Reports)
A narrative rewrite translating the PwC Strategy& Crypto Survey 2025 (a 2,500-respondent industry report) for general retail investors. Includes audience analysis, narrative rewrite, and ethical reflection on the risks of communicating selectively-sampled survey data.

**Tools:** Microsoft Word, audience analysis framework

### [Titanic Analysis Presentation](./Presentations)
Seven-slide presentation distilling the Titanic survival analysis into a concise visual format. Demonstrates slide design principles including sandwich structure, consistent visual motifs, and native editable charts.

**Tools:** Microsoft PowerPoint

### [SQL Practice Queries](./SQL)
Annotated SQL queries demonstrating foundational analytics patterns: filtering, sorting, aggregation, grouping, and joining tables.

**Tools:** SQLite
