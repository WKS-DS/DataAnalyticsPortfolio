# Tableau Projects

## HR Analytics Dashboard

Interactive dashboard analyzing employee demographics, attrition, and department-level patterns using a multi-table HR dataset.

**Live Dashboard:** [View on Tableau Public](https://public.tableau.com/app/profile/will.scott8637/viz/HR_Tableau_17823693965210/Q3?publish=yes)

**Visuals Included:**
- Employee count by department
- Gender distribution
- Employee age distribution
- Salary analysis by department

**Data Source:** HR_Employees dataset with three related tables (Employees, Departments, Attrition Records)

**Tools:** Tableau Public, data modeling with star schema relationships, calculated fields, grouped dimensions

**Files in this folder:**
- `HR_Tableau.twb` — Tableau workbook file (XML format, requires original dataset to run)
