# Power BI Projects

Interactive business intelligence dashboards built with Microsoft Power BI.

## Project: Sales Performance Dashboard

### Objective
Build an interactive sales reporting dashboard that lets stakeholders filter performance data by product category and customer type, with at-a-glance KPI summaries and detailed transaction-level visibility.

### Dataset
Sales transactions dataset covering product information, customer segmentation (new vs. returning), regional distribution, sales representatives, and quarterly performance across the 2023 calendar year.

### Tools
- Microsoft Power BI (browser-based service)
- Slicers for interactive filtering
- KPI card visuals
- Bar charts and data tables

### Dashboard Components
- **Total Sales KPI Card:** Displays $5.02M aggregated sales amount
- **Product Category Slicer:** Filters the report by product line (Clothing, Furniture, Electronics, Food)
- **Customer Type Slicer:** Filters between new and returning customers
- **Sales Report Table:** Detailed transaction-level data showing product, customer type, category, quantity sold, region, sales rep, and time dimensions
- **Sales by Product Category Bar Chart:** Visual comparison of sales performance across product lines

### Findings
- Total tracked sales reached $5.02M across all categories in 2023.
- Clothing led product categories by sales amount, followed closely by Furniture and Electronics.
- The dashboard supports drill-down analysis by region, sales rep, quarter, and customer segment.
